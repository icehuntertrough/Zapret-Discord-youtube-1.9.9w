## 📖 О проекте

**zapret-discord-youtube** (также известный как **zapret-discord**, **DSFix**, **discord-youtube** и **zapretds**) — это удобная сборка **zapret** для Windows, объединяющая готовые стратегии обхода DPI, автоматическую настройку и инструменты диагностики.

Проект позволяет быстро восстановить стабильный доступ к популярным сервисам без необходимости вручную настраивать **WinDivert** и **winws**.

## ⚡ Быстрый старт

1. Скачайте последнюю версию из **Releases**.
2. Распакуйте архив в папку без кириллицы.
3. При необходимости включите **Secure DNS**.
4. Запустите подходящий `general*.bat`.
5. Если стратегия подошла — установите её в автозапуск через `service.bat`.

---

## 💬 Discord

Для повышения стабильности работы Discord рекомендуется обновить файл **Hosts**.

После обновления можно получить:

- ⚡ более быструю загрузку клиента;
- 📂 ускоренное скачивание файлов;
- 💬 более быструю загрузку сообщений;
- 🖼️ стабильную загрузку изображений и другого медиаконтента.

## ▶️ YouTube

Для корректной работы рекомендуется:

- включить **Secure DNS**;
- попробовать несколько стратегий;
- при необходимости отключить блокировщики рекламы.

## 🛡️ Безопасность

В проекте используется **WinDivert** — драйвер фильтрации сетевого трафика.

Некоторые антивирусы могут определять его как:

- `WinDivert`
- `RiskTool`
- `PUA`

Это связано с особенностями работы драйвера и не означает наличие вредоносного кода.
